## 🚀 Featured Projects

### 🏛️ [Lokarya — Gamified Civic Engagement Platform](https://github.com/ashishwankhade/Lokarya-A-Gamified-Civic-Engagement-Platform)

> Full-stack civic tech platform · React · Node.js · MongoDB · FastAPI · Google Gemini 2.5 Flash

- 9-step complaint pipeline with GPS-based Haversine deduplication and automated SLA escalation
- Multi-role RBAC across 5 user types with JWT dual-token auth and Google OAuth 2.0
- XP gamification engine with configurable rules, badges, leaderboard, and reward redemption
- AI Civic Assistant powered by **Google Gemini 2.5 Flash** via FastAPI + LangChain backend
- 3-layer QR attendance gate (token + GPS proximity + registration check) for community missions
- WhatsApp/SMS integration via Twilio with magic-link field worker proof upload

`React.js` `Node.js` `MongoDB` `FastAPI` `Python` `Gemini 2.5` `JWT` `Twilio` `Cloudinary` `TailwindCSS`
